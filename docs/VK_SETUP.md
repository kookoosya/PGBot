# Как получить API ВКонтакте — пошаговая инструкция

Эта инструкция для тех, кто не программист. Просто следуйте шагам.

## Что понадобится

- Аккаунт ВКонтакте
- Сообщество (группа) для бота — можно создать новое
- Сайт с HTTPS (для Callback API) — после деплоя

---

## Шаг 1. Создайте сообщество

1. Откройте [vk.com](https://vk.com)
2. Слева нажмите **Сообщества** → **Создать сообщество**
3. Тип: **Тематическое** или **Публичная страница**
4. Название: **Народный Контроль Пушкинские Горы**
5. Загрузите аватар (можно перо 🪶 на зелёном фоне)

---

## Шаг 2. Включите сообщения

1. Зайдите в своё сообщество → **Управление**
2. Раздел **Сообщения** → включите **Сообщения сообщества**
3. Включите **Возможности ботов**

---

## Шаг 3. Callback API (главное!)

1. **Управление** → **Работа с API** → вкладка **Callback API**
2. Нажмите **Настройки сервера**
3. **Адрес сервера**: `https://ВАШ-ДОМЕН.ru/api/v1/vk/callback`
   - Пока сайт не задеплоен — оставьте пустым, настроите позже
4. Нажмите **Подтвердить** — ВК покажет **строку подтверждения** (например `a1b2c3d4`)
5. Скопируйте её в `.env`:
   ```
   VK_CONFIRMATION_CODE=a1b2c3d4
   ```
6. Перезапустите backend и снова нажмите **Подтвердить** в ВК

---

## Шаг 4. Секретный ключ

1. На той же странице Callback API найдите **Секретный ключ**
2. Скопируйте в `.env`:
   ```
   VK_SECRET_KEY=ваш_секретный_ключ
   ```

---

## Шаг 5. Ключ доступа (токен)

1. **Управление** → **Работа с API** → **Ключи доступа**
2. Нажмите **Создать ключ**
3. Отметьте права:
   - ✅ Управление сообществом
   - ✅ Сообщения сообщества
   - ✅ Фотографии (для вложений)
4. Скопируйте ключ в `.env`:
   ```
   VK_GROUP_TOKEN=vk1.a.ваш_токен...
   ```

---

## Шаг 6. Включите события

1. **Callback API** → **Типы событий**
2. Включите: **Входящее сообщение** (`message_new`)
3. Сохраните

---

## Шаг 7. Проверка

1. Напишите боту в личные сообщения сообщества: **Привет**
2. Должен ответить приветствие в пушкинском стиле с кнопками

---

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| «Сервер не отвечает» | Проверьте, что сайт доступен по HTTPS и backend запущен |
| «Неверный код подтверждения» | `VK_CONFIRMATION_CODE` в `.env` должен совпадать с ВК |
| Бот не отвечает | Проверьте `VK_GROUP_TOKEN` и что событие `message_new` включено |
| Ошибка secret | `VK_SECRET_KEY` в `.env` должен совпадать с ВК |

## Ссылка на бота

После настройки дайте жителям ссылку:
```
https://vk.com/im?sel=-ID_СООБЩЕСТВА
```

ID сообщества виден в адресе: `vk.com/club123456789` → ID = `123456789`
